# 🌱 Eco-Efficient Path Planner

An intelligent route optimization system that helps users find the most fuel-efficient and environmentally friendly travel path between locations.

---

## 📌 Project Description

The Eco-Efficient Path Planner is designed to provide optimized routes by considering:

- Shortest distance
- Traffic conditions
- Fuel consumption
- Carbon emissions
- Travel time

The project aims to promote sustainable transportation and reduce environmental impact through smart path planning.

---

## 🚀 Features

- ✅ Shortest Path Optimization
- ✅ Eco-Friendly Route Selection
- ✅ Traffic-Based Navigation
- ✅ Fuel Consumption Estimation
- ✅ Carbon Emission Calculation
- ✅ Interactive Dashboard
- ✅ Real-Time Route Visualization
